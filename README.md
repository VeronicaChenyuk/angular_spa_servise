# angular_spa_servise
Небольшое SPA приложение для сортировки данных с сервера.



## Getting Started

cd angular_spa_servise
cd my-app
npm install
npm start

## MVP Functionality
* Рабочий стол, на котором расположены виджеты с набором данных.
* Форма для создания новых виджетов.
* Фильтры для виджетов.
